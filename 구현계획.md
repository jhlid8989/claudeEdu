# 클로드 교육 웹 페이지 구현 계획

## 개요

`교육요약.md` 파일(6개 챕터 + 개념 연결 지도)을 시각적으로 풍부한 단일 랜딩 페이지로 변환한다.
스크롤 기반 인터랙션과 배경 파티클 효과를 중심으로, 교육 내용을 인상적으로 전달하는 것이 목표다.

---

## 페이지 구조 (8개 섹션)

### Hero 섹션
- 전체 화면 높이 (`h-screen`)
- tsParticles 배경: AI 네트워크 연결 선 파티클
- 제목 "클로드 교육 핵심 개념" — 글자 하나씩 순서대로 나타나는 stagger 애니메이션
- 태그 뱃지 (PRD, MCP 서버, MVP 등 7개)
- 스크롤 유도 화살표 (바운스 애니메이션)

### Chapter 01 — PRD 섹션
- 8가지 구성요소 카드 그리드 (스크롤 시 순서대로 슬라이드인)
- 좋은 PRD vs 나쁜 PRD 좌우 비교 (행 단위 애니메이션)
- PRD가 없을 때 문제 상황 4개 역할 카드
- 배경: 진한 슬레이트 (`slate-900`)

### Chapter 02 — MCP 서버 섹션
- 설치→연결→사용 3단계 플로우 카드 (순서대로 나타남)
- JSON 설정 예시 코드블록 (react-syntax-highlighter 구문 강조)
- 읽기/실행/쓰기 기능 카드 3개 (hover 확대 효과)
- 배경: 슬레이트 950

### Chapter 03 — MVP 섹션
- 잘못된/올바른 이해 좌우 비교 카드 (슬라이드인 애니메이션)
- MVP 결정 기준 YES/NO 다이어그램
- 배달 앱 MVP 예시 테이블 (행마다 순서대로 등장)
- PRD↔MVP 관계 설명 카드
- 배경: 인디고 950 → 슬레이트 900 그라디언트

### Chapter 04 — Sub-agent / Agent Team 섹션
- Sub-agent, Agent Team, Task Manager 개념 3개 카드 (순서대로 등장)
- 세 개념의 관계도 시각화 (스태거 애니메이션)
- 배경: 아연 950

### Chapter 05 — Task Manager 비교 섹션
- Shrimp vs Claude 11개 항목 비교 테이블 (행마다 등장)
- 언제 무엇을 쓸까 2개 선택 카드
- 배경: 스톤 950 → 슬레이트 900 그라디언트

### Chapter 06 — Supabase vs Notion 섹션
- 한 줄 정의 좌우 카드 (슬라이드인 애니메이션)
- 용도 비교 테이블 (체크/X 아이콘 스케일인 spring 애니메이션)
- 함께 쓰는 조합 설명
- 배경: 에메랄드 950 → 틸 950 그라디언트

### 전체 개념 연결 지도 섹션
- PRD → MVP → 개발 실행 인터랙티브 플로우 다이어그램
- 각 노드 클릭 시 해당 섹션으로 스크롤 이동
- 화살표 드로잉 애니메이션 (scaleY)
- 배경: 코스믹 래디얼 그라디언트

| 요소 | 설명 |

## 구현 순서

1. **프로젝트 초기화** — `create-next-app` TypeScript + Tailwind + App Router
2. **패키지 설치** — framer-motion, lenis, tsparticles, react-syntax-highlighter
3. **globals.css** — 다크 테마, 커스텀 스크롤바
4. **공통 레이아웃** — LenisProvider, Navbar, ProgressBar
5. **Hero 섹션** — 파티클 배경 + 타이틀 애니메이션
6. **컨텐츠 섹션 1~6** — 섹션별 순서대로 구현
7. **개념 연결 지도** — 인터랙티브 플로우 + 앵커 링크
8. **빌드 검증** — `npm run build` TypeScript 오류 없음 확인
