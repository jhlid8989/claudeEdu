# Vercel 배포 방법

## 배포 플랫폼: Vercel

Next.js 공식 호스팅 플랫폼으로, GitHub 연동 시 코드 push 시마다 자동 재배포된다.
무료 플랜에서 개인 프로젝트 배포가 충분히 가능하다.

---

## 사전 준비

- GitHub 계정
- Vercel 계정 (GitHub 계정으로 가입 가능)
- Node.js 18+ 설치 확인

---

## 방법 1: GitHub 연동 배포 (권장)

### 1단계 — GitHub 저장소 생성 및 push

```bash
# 프로젝트 폴더에서 실행
cd "C:\Users\ktnet\Desktop\클로드 교육\claude_education"

# git 초기화 (이미 되어 있으면 생략)
git init

# 파일 스테이징
git add .

# 첫 커밋
git commit -m "feat: 클로드 교육 웹 페이지 초기 구현"

# GitHub에서 새 저장소 생성 후 연결
git remote add origin https://github.com/[내_계정명]/claude-education.git

# push
git branch -M main
git push -u origin main
```

### 2단계 — Vercel에서 프로젝트 연결

1. [vercel.com](https://vercel.com) 접속 → 로그인
2. **"Add New Project"** 클릭
3. GitHub 저장소 목록에서 `claude-education` 선택
4. 설정 확인:
   - Framework Preset: **Next.js** (자동 감지)
   - Root Directory: `./`
   - Build Command: `next build` (기본값)
   - Output Directory: `.next` (기본값)
5. **"Deploy"** 클릭

### 3단계 — 배포 완료

- 빌드 시간: 약 1~2분
- 배포 URL 자동 발급: `https://claude-education-[랜덤].vercel.app`
- 이후 `main` 브랜치에 push 할 때마다 자동 재배포

---

## 방법 2: Vercel CLI 배포 (빠른 방법)

```bash
# Vercel CLI 설치
npm install -g vercel

# 프로젝트 폴더에서 실행
cd "C:\Users\ktnet\Desktop\클로드 교육\claude_education"

# 배포 (처음 실행 시 로그인 및 프로젝트 설정)
vercel

# 프로덕션 배포
vercel --prod
```

CLI 실행 후 물어보는 항목:
- `Set up and deploy?` → **Y**
- `Which scope?` → 본인 계정 선택
- `Link to existing project?` → **N** (처음이면)
- `What's your project's name?` → `claude-education` 입력
- `In which directory is your code located?` → `./` (기본값)

## 로컬 빌드 확인 (배포 전 검증)

```bash
# 프로덕션 빌드 생성
npm run build

# 빌드 결과 로컬에서 실행
npm run start

# 브라우저에서 http://localhost:3000 확인
```

---

## 배포 후 커스텀 도메인 연결 (선택)

1. Vercel 프로젝트 → **Settings** → **Domains**
2. 보유한 도메인 입력 (예: `claude-education.com`)
3. DNS 설정: 도메인 등록 업체에서 CNAME 레코드 추가
   ```
   CNAME  www  cname.vercel-dns.com
   ```

## 주요 Vercel 무료 플랜 제한

| 항목 | 무료 플랜 |
|------|----------|
| 프로젝트 수 | 무제한 |
| 대역폭 | 월 100GB |
| 빌드 시간 | 월 6,000분 |
| 서버리스 함수 실행 | 월 100GB-시간 |
| 커스텀 도메인 | 무제한 |
| HTTPS | 자동 적용 |

이 프로젝트는 정적 사이트(SSG)로 빌드되므로 무료 플랜으로 충분히 운영 가능하다.
